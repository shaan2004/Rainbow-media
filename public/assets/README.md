# BillIT NOW Product Assets Directory

Place your actual software screenshot files in this directory to display them inside the **Reports Showcase** mockup block:

- **Invoice Summary Screen**: Name it `r1.png` (or `r1.jpg` / `r1.webp`)
- **Invoice Detail Screen**: Name it `r2.png`
- **Staff Report Screen**: Name it `r3.png`
- **Daywise Report Screen**: Name it `r4.png`
- **Monthwise Report Screen**: Name it `r5.png`

The website automatically detects these image assets on load and transitions from the custom HTML mockups into your actual screenshots!
